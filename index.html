Actions are great as they allow a node's properties to be manipulated but they are weightless. They perform the same from start to finish, luckily there are tons of easing methods to help overcome this and make the actions look more dynamic and alive. Though the easing methods can be used on virtually any action, this page will be using them with a MoveBy action.
